# Plan: Text Brightness, Bold, Shadows & Section Transitions

## Problem
Multiple homepage sections have text that is too dim/faint. The "WhyReverbex", "Methodology", "ReverbexBond", "FAQ", and "CaseStudyClient" sections use very low opacity values (`text-white/30`, `text-white/35`, `text-white/40`) making text hard to read. The user wants brighter, bolder text with proper shadows and smoother section transitions.

## Section Transitions (between sections on homepage)

The homepage already has good GSAP scroll animations within each section. To add **inter-section transitions**, add staggered entrance animations to each section wrapper in `app/page.tsx`:

### In each section component (WhyReverbex, Methodology, ReverbexBond, FAQSection, ContactSection):
- Add a section wrapper class `section-reveal` with CSS:
```css
.section-reveal {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.8s cubic-bezier(0.16,1,0.3,1),
              transform 0.8s cubic-bezier(0.16,1,0.3,1);
}
.section-reveal.visible {
  opacity: 1;
  transform: translateY(0);
}
```
- Use IntersectionObserver (or leverage existing GSAP ScrollTrigger) to trigger `.visible` class when section enters viewport.

**Alternative (simpler, already partially implemented):** Most sections already use `gsap.fromTo` with `scrollTrigger: { start: "top 80%" }` which fades/slides sections in. The existing animations are sufficient — the main fix is text brightness.
